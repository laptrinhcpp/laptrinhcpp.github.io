Ta còn có thêm một cách nữa là dùng tham số, đối số của hàm.

Tham số và đối số của hàm

Trong nhiều trường hợp, bạn cần đưa giá trị vào hàm để có thể làm việc với giá trị đó. Ví dụ bạn viết hàm cộng giá trị của hai số thì bạn phải đưa hai số đó vào hàm, nếu không thì sao nó biết cộng hai số nào? Bạn có thể làm điều đó thông qua tham số và đối số.

Tham số hàm là một biến được sử dụng trong hàm. Chúng được định nghĩa ngay trong dấu ngoặc đơn ‘()’ sau tên hàm. Nếu có nhiều tham số thì chúng cách nhau bởi dấu phẩy.

void doPrint()             // hàm không có tham số
{
    std::cout << "In doPrint()" << '\n';
}
 
void printValue(int x)     // hàm có một tham số là biến x
{
    std::cout << x  << '\n';
}
 
int add(int x, int y)       // hàm có hai tham số là biến x và y
{
    return x + y;
}

Đối số chính là giá trị được đưa vào hàm khi gọi hàm. Ví dụ:

1
2
3
doPrint();         // không có đối số
printValue(6);     // 6 là đối số được đưa vào hàm
add(2, 3);         // 2 và 3 là các đối số được đưa vào hàm

Lưu ý: Các tham số và đối số cách nhau bởi dấu phẩy.

Cách hoạt động của tham số và đối số

Khi hàm được gọi, tất cả các tham số trong hàm được tạo ra. Sau đó các tham số copy giá trị của đối số tương ứng. Quá trình này được gọi là truyền giá trị (pass by value).

#include <iostream>
 
void printValues(int x, int y)   // hàm này có hai tham số là x và y
{
    std::cout << x << '\n';
    std::cout << y << '\n';
}
 
int main()
{
    printValues(6, 7);           // hàm được gọi và truyền đối số 6 và 7 vào hàm 
 
    return 0;
}

Khi hàm printValue được gọi, biến x được tạo ra và gán giá trị 6, biến y được tạo ra và gán giá trị 7.

Kết quả:

6
7

Lưu ý: Không được phép nhận tham số và đối số với số lượng khác nhau, chẳng hạn như hàm có 2 tham số, nhưng lại nhập 3 đối số vào. Đối số không nhất thiết là literal, nó có thể là một biểu thức bất kì miễn sao giá trị biểu thức đó có kiểu dữ liệu giống tham số tương ứng.

    return 0;

Đầu tiên, biến num sẽ được khởi tạo với giá trị người dùng nhập vào. Tiếp theo, hàm printDouble được gọi. Ta truyền giá trị của num vào hàm printDouble và giá trị đó được gán cho biến value. Cuối cùng, chương trình in giá trị gấp đôi value ra màn hình.

int multiply(int z, int w)
{
    return z * w;
}

{
    std::cout << add(4, 5) << '\n';               // x = 4, y = 5, x + y = 9
    std::cout << add(1 + 2, 3 * 4) << '\n';       // x = 3, y = 12, x + y = 15
 
    int a{ 5 };
    std::cout << add(a, a) << '\n';               // a + a = 5 + 5 =  10
