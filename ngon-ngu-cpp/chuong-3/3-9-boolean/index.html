3.9 Boolean

Trong đời sống hằng ngày, chắc hẳn bạn sẽ gặp những câu nghi vấn dạng "đúng hay sai", "có hay không". "C++ có khó không?" - "Có", "Chị hiểu hông?" - "Không"

Bây giờ ta xét các câu khẳng định mà có thể đúng (true) hoặc sai (false), ví dụ "Hiệp đẹp trai", câu khẳng định trên là true, "Hiệp là con gái", câu khẳng định trên hoàn toàn là false.

Những câu ví dụ trên chỉ có hai khả năng trả lời là "yes/true" hoặc "no/false". Và hai cách trả lời true hoặc false đó được gọi là boolean.

Biến boolean

Biến boolean là loại biến chỉ có khả năng chứa hai giá trị là true hoặc false.

Để định nghĩa một biến boolean, ta dùng từ khoá bool.

1
bool b;

Để khởi tạo hoặc gán giá trị true, false cho biến boolean, ta dùng từ khoá true và false.

1
2
3
4
bool b1 { true };
bool b2 { false };
b1 = false;
bool b3{}; // mặc định là false

Giống như toán tử trừ đơn phân (-) có thể chuyển đổi một số dương thành số âm và ngược lại, toán tử not logical (!) có thể chuyển đổi giá trị true thành false và false thành true.

1
2
3
//
bool b1 { !true };  // b1 sẽ được gán giá trị false
bool b2 { !false }; // b2 sẽ được gán giá trị true

Như bạn đã biết, dữ liệu trong bộ nhớ chỉ có thể lưu dưới dạng nhị phân tức là chỉ gồm các số 0 và 1. Do đó, thứ được lưu trong biến boolean không thể là chữ "true" hay "false" được. Thay vào đó, nó sẽ được lưu dưới dạng integer: true sẽ được lưu thành 1 và false sẽ được lưu thành 0. Đó cũng là lý do boolean thuộc loại integral, bản chất boolean cũng chỉ là integer (gồm giá trị 1 (true) và false (0)).

In ra boolean

Khi ta in giá trị của boolean thông qua std::cout, nó sẽ in ra 1 nếu là true và 0 nếu là false.

#include <iostream>
 
int main()
{
    std::cout << true << std::endl;  // true là 1
    std::cout << !true << std::endl; // !true là 0
 
    bool b{false};
    std::cout << b << std::endl;     // b là false nên in ra 0
    std::cout << !b << std::endl;    // !b là true nên in ra 1
    return 0;
}

Output:

1
0
0
1

Hàm có trả về boolean

Tương tự như integer, boolean cũng có hàm trả về tương ứng. Nó trả về giá trị 0 (false) hoặc 1 (true). Ta dùng loại hàm này khi kiểm tra điều gì đó có đúng hay không.

#include <iostream>
 
// trả về true nếu x bằng y và trả về false nếu chúng khác nhau
bool isEqual(int x, int y)
{
    return (x == y); // toán tử == so sánh hai giá trị, nếu bằng nhau thì trả về true, nếu khác nhau thì trả về false
}
 
int main()
{
    std::cout << "Enter an integer: ";
    int x{};
    std::cin >> x;
 
    std::cout << "Enter another integer: ";
    int y{};
    std::cin >> y;
 
    std::cout << std::boolalpha; 
    
    std::cout << x << " and " << y << " are equal? ";
    std::cout << isEqual(x, y);
 
    return 0;
}

Dưới đây là output cho hai trường hợp:

Enter an integer: 5
Enter another integer: 5
5 and 5 are equal? true
Enter an integer: 6
Enter another integer: 4
6 and 4 are equal? false

Nó hoạt động như thế nào? Đầu tiên, nó nhận hai giá trị x, y từ người dùng. Khi gặp gọi hàm isEqual(x, y), compiler biên dịch hàm isEqual. Trong hàm này, nó so sánh x và y. Ở trường hợp thứ nhất x=5, y=5 nên x=y, biểu thức x==y đúng nên nó trả giá trị true về hàm main. Kết quả là in ra "true". Ở trường hợp thứ hai, x=6, y=4 nên x không bằng y, biểu thức x==y sai nên nó trả giá trị false về hàm main. Kết quả là in ra "false".

Kí kiệu == được gọi là toán tử so sánh, nó mang ý nghĩa là "giá trị bên vế trái bằng giá trị bên vế phải" và sẽ trả về true nếu khẳng định đó đúng, false nếu khẳng định đó sai.
