3.2 Void

Void là kiểu dữ liệu dễ hiểu nhất. Về cơ bản, void có nghĩa là "không có kiểu dữ liệu".

Do đó, bạn không thể tạo biến có kiểu void được.

1
void value; // không thể làm như thế này được

Tạo biến kiểu void nó giống như bắt bạn xây một căn nhà mà không có nguyên vật liệu vậy. Nó hoàn toàn không có gì hết.

Vậy thì nó được dùng để làm những việc gì?

Hàm không trả về giá trị

Phần lớn các trường hợp void được dùng để tạo một hàm không trả về như bạn đã được học.

1
2
3
4
5
void writeValue(int x)
{
    std::cout << "The value of x is: " << x << '\n';
    // không có giá trị trả về
}

Biểu thị hàm không cần tham số

Trong ngôn ngữ C, void được đặt trong dấu ngoặc tròn () của hàm mang ý nghĩa hàm này không có tham số.

1
2
3
4
5
6
int getValue(void) // void ở đây mang ý nghĩa hàm này không có tham số
{
    int x;
    std::cin >> x;
    return x;
}

Tuy nhiên trong C++, bạn không cần làm vậy cũng được.

1
2
3
4
5
6
int getValue() // viết như thế này là được
{
    int x;
    std::cin >> x;
    return x;
}

Ngoài ra, void còn có thêm một chức năng nữa nhưng nó liên quan đến con trỏ nên các bạn sẽ được học sau khi học con trỏ.
